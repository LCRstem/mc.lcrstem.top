Minecraft Web Edition

基于现代Web技术的网页版Minecraft

链接在网页底部

简介

通过JavaScript等技术，Eaglercraft团队实现了在浏览器中直接运行Minecraft Java Edition。无需安装任何客户端，在临时的设备上也能便携游玩，支持多版本和JavaScript和WebAssembly两种高性能运行模式

技术原理

我们采用前沿的Web技术栈，确保游戏运行的流畅性和稳定性：

JS
JavaScript核心
WA
WebAssembly编译
GL
WebGL渲染
VM
TeaVM虚拟机
SV
本地式存储
